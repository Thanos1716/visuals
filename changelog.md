# Visual Armor Trims Changelog

## Visual Armor Trims v1.1

This release significantly reduced the number of textures by making use of `paletted_permutations`, as well as fixed a couple of texture issues.

### Additions:
- New Pack Icon
- Use `paletted_permutations`

### Fixes:
- Helmet Cutout (now uses correct texture for all items)

## Visual Armor Trims v1.0

Add support for all 1.20.X armor trim textures

### Additions:
- Default Trim:
  - Coast
  - Dune
  - Eye
  - Host
  - Raiser
  - Rib
  - Sentry
  - Shaper
  - Silence
  - Snout
  - Spire
  - Tide
  - Vex
  - Ward
  - Wayfinder
  - Wild
- Separate Trim Textures:
  - Turtle Helmet
  - Netherite Boots
